# BPSR-AutoModules_JP

このリポジトリは、以下の参照元プロジェクトについて、その内容と意図をリスペクトしながら日本語版として調整したものです。

## 参照元

- [mrsnakke/BPSR-AutoModules](https://github.com/mrsnakke/BPSR-AutoModules)

## 方針

- 参照元プロジェクトの構成、機能、考え方を尊重しています。
- 本リポジトリでは、主に UI・文言・説明の日本語化を目的として変更しています。
- ライセンス、著作権、クレジットは参照元を尊重してください。

## 使用方法

1. パケット取得には `Npcap` が必要です。配布物には `npcap-1.83.exe` を同梱していないため、事前に [Npcap 公式サイト](https://npcap.com/) からインストールしてください。
2. アプリを起動します。
3. 使用するネットワークインターフェースを選択します。
4. モジュール種別と必要な属性を選択します。
5. 必要に応じてプリセットを選択します。
6. `監視開始` を押します。
7. ゲーム内でチャンネル変更やキャラクター切替などを行い、モジュールデータを取得します。
8. `監視停止` を押して、取得したモジュール一覧を保持します。
9. 条件を調整して `最適化開始` を押します。
10. 必要に応じて `CSV保存` でモジュール一覧を書き出すか、`CSV読込` で保存済み一覧を読み込みます。

## CSV 機能

- `CSV保存` を使うと、監視で取得したモジュール一覧を CSV ファイルとして保存できます。
- 保存した CSV は、次回以降に `CSV読込` から読み込めます。
- `CSV読込` で読み込んだモジュール一覧は、ゲーム内で再度取得しなくてもそのまま `最適化開始` で再計算できます。
- 条件だけを変えて何度も最適化を試したい場合は、先に CSV へ保存しておくと再取得の手間を減らせます。
